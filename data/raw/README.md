# Datos brutos

Esta carpeta contiene la versión bruta del conjunto de datos que se utiliza como punto de partida para el Trabajo de Fin de Grado.

## Nombre del conjunto de datos

`Eixample_03_2020_2025.csv`

## Descripción

El archivo contiene mediciones horarias de ozono troposférico (O3) registradas en la estación de calidad del aire del Eixample de Barcelona para el periodo 2020-2025.

La variable objetivo del proyecto es la concentración horaria observada de O3.

## Fuente

Los datos proceden de la Red de Vigilancia y Previsión de la Contaminación Atmosférica de Catalunya, XVPCA, publicada por la Generalitat de Catalunya.

## Periodo temporal

- Inicio previsto: 2020.
- Fin previsto: 2025.

La cobertura temporal exacta se comprobará durante la fase de análisis exploratorio de datos.

## Uso dentro del proyecto

Este archivo contiene los datos brutos sobre los que se sustenta el TFG y, por tanto, no debe modificarse manualmente.

De ser necesario realizar transformaciones para el análisis del conjunto de datos, sus resultados se guardarán en la ruta `data/processed/`.
